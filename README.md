# CV-ChessAutomatic
체스보드 자동 검색을 위한 프로그램

# 필요한 것
* opencv-python

# 설치
```bash
pip3 install opencv-python
```

# 쓰는 방법
```bash
python3 CV-chess.py [your chess image file] #ex)python3 CV-chess.py ./chess.jpg
```
그리고 설명을 따르고 진행해주세요.

# 결과
파일의 포맷은 아래와 같습니다.
```bash
img_[chess height]x[chess width].[file extension] #ex)img_7x10.jpg
```
